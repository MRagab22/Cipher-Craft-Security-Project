    Tutorial: How the Algorithms Work

    • Caesar Cipher: Shifts each letter in the text by a fixed number of positions in the alphabet.
      For example, a shift of 3 turns 'A' into 'D' and 'B' into 'E'.
    • Monoalphabetic Cipher: Replaces each letter with another based on a fixed 26-letter key.
      The key must contain all 26 letters in a unique order.
    • Playfair Cipher: Encrypts pairs of letters using a 5x5 grid created from a keyword.
      Letters are substituted based on their positions in the grid.
    • Transposition Cipher: Rearranges letters by writing them in rows and reading by column order.
      A numeric key (e.g., 5 4 2 3 1) specifies the column sequence.
    • Vigenère Cipher: Shifts letters using a repeating keyword, varying the shift for each letter.
      Each letter of the key determines a different Caesar shift.
    • Rail Fence Cipher: Writes text in a zigzag pattern across rows and reads it row by row.
      The number of rows (rails) defines the zigzag's depth.
